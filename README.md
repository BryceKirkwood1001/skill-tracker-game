# Skill Tracker

A Python application that gamifies learning new skills.

To use the app, run database_setup.py first to create the database used to store your goals.

## Current Features

- Track skills and progress
- XP and leveling system
- Basic user profiles
- JSON persistence (for user data)
- SQLite database

## Planned Features

- Achievements
- Collectibles
- Web UI
- AI-powered goal setting

Please note that the project is currently not ready to be distributed or used by anyone outside of any developers
